Get Text

Please use Safari on macOS 12 (Monterey) or higher. Otherwise this page might not work properly.

  1. Copy the image
  2. Click in the input field
  3. Paste the image
  4. Move your cursor to the text within the image
  5. Wait a few seconds until the cursor changes to a text selector
  6. Select the text from the image using the cursor
